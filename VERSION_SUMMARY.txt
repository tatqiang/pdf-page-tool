   Total Combinations: 901 options

================================================================================
                          SYSTEM REQUIREMENTS
================================================================================

REQUIRED
--------
- Windows Operating System
- Python 3.x
- PyMuPDF (fitz) library

INSTALLATION
------------
1. Install Python from https://www.python.org/downloads/
   (Check "Add Python to PATH" during installation)
2. Install PyMuPDF: pip install PyMuPDF
3. Done!

TROUBLESHOOTING
---------------
If "Python not found" error:
1. Run check_python.bat for diagnosis
2. Follow the provided solutions
3. See README.txt for detailed troubleshooting

================================================================================
                              SUPPORTED TAGS
================================================================================

| Syntax       | Display    | Example     |
|-------------|-----------|-------------|
| {{p-x}}     | x         | 5           |
| {{p-x/y}}   | x / y     | 5 / 28      |
| {{p-xofy}}  | x of y    | 5 of 28     |
| {{pg-x}}    | Page x    | Page 5      |
| {{pg-x/y}}  | Page x/y  | Page 5/28   |
| {{pg-xofy}} | Page x of y | Page 5 of 28 |

Use these tags in your document headers/footers before converting to PDF.
