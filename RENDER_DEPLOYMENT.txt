TROUBLESHOOTING
================================================================================

If deployment fails:

1. Check Build Logs in Render dashboard
   - Click "Logs" tab
   - Look for error messages

2. Common issues:
   
   ‚ùå "requirements.txt not found"
   ‚Üí Make sure requirements.txt is in root folder
   
   ‚ùå "Module not found: flask"
   ‚Üí Check requirements.txt has correct packages
   
   ‚ùå "Application failed to start"
   ‚Üí Check Procfile has correct command
   ‚Üí Verify web_app.py has no syntax errors
   
   ‚ùå "Port binding failed"
   ‚Üí Make sure app.run() uses port from environment
   ‚Üí Render provides PORT automatically

3. Check application logs:
   - Click "Logs" tab in Render
   - Look for Python errors
   - Check if Flask app started correctly

4. Test locally first:
   cd c:\pwa\program\pdf_page
   python web_app.py
   
   If it works locally but not on Render:
   - Check all files are uploaded to GitHub
   - Verify requirements.txt has all dependencies
